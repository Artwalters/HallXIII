# Fonts voor Hall XIII

Deze folder moet de volgende font bestanden bevatten:

## Benodigde Fonts

### 1. SCHABO Condensed
- `schabo-condensed.woff2`
- `schabo-condensed.woff`
- Gebruikt voor: Grote headings en titels

### 2. Relost Brush
- `relost-brush.woff2`
- `relost-brush.woff`
- Gebruikt voor: Stap indicatoren en handgeschreven tekst

### 3. GRIFTER
- `grifter.woff2`
- `grifter.woff`
- Gebruikt voor: Navigatie en body tekst

## Hoe te installeren

1. Plaats de font bestanden in deze folder (`public/fonts/`)
2. Zorg ervoor dat de bestandsnamen exact overeenkomen met bovenstaande namen
3. De fonts worden automatisch geladen via `app/globals.css`

## Waar de fonts te verkrijgen

- Zorg ervoor dat je de juiste licentie hebt voor deze fonts
- Converteer indien nodig TTF/OTF bestanden naar WOFF en WOFF2 formaat
- Online tools zoals [Transfonter](https://transfonter.org/) kunnen helpen bij conversie
